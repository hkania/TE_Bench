_Scaffolding of the `output/model_data` directory is included for users hoping to use Simulation_Snakefile._

## Subdirectory `Dfam/`
_The subdirectory Dfam is where users should download Dfam.embl files. This step is now automated within the Simulation_Snakefile, configured with Dfam-curated_only-1.embl, with the option to change the embl. This is where downstream scripts expect those files to be._

## Subdirectory `RepBase/`
_The subdirectory RepBase is where users should download RepBase.embl files. This is where downstream scripts expect those files to be. See [instructions]() for more information._
